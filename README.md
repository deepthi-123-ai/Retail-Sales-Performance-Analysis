# Retail-Sales-Performance-Analysis
## Project Overview

This project analyzes retail sales data using Python. The goal is to understand sales performance, customer behavior, and product category trends through data analysis and visualization.

## Tools Used

* Python
* Pandas
* Matplotlib
* Google Colab

## Dataset

The dataset contains retail transaction records with the following information:

* Transaction ID
* Date
* Customer ID
* Gender
* Age
* Product Category
* Quantity
* Price per Unit
* Total Amount

## Analysis Performed

* Data Cleaning and Exploration
* Total Sales Calculation
* Product Category Analysis
* Gender-wise Sales Analysis
* Monthly Sales Trend Analysis
* Top Customer Analysis
* Data Visualization using Charts

## Key Findings

* Total Sales: ₹7,700,855
* Total Orders: 1000
* Best Selling Category: Sports
* Highest Sales Month: December
* Male customers generated slightly higher sales than female customers.
* Customer ID 1098 was the top customer.

## Conclusion

The analysis shows that Sports products generated the highest revenue. Sales were highest in December, indicating increased customer purchases during that period. The project demonstrates how data analytics can be used to gain valuable business insights and support decision-making.
